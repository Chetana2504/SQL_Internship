# Hospital Management Database Project

##`   Overview
This project is part of the SQL Developer Internship Task 1.  
It represents a Hospital Management System created using MySQL.

## Tables
- Department  
- Doctor  
- Patient  
- Appointment  
- Bill  

## Relationships
- One Department → Many Doctors  
- One Doctor → Many Appointments  
- One Patient → Many Appointments  
- One Appointment → One Bill  

## Concepts Used
- DDL Commands  
- Primary and Foreign Keys  
- Normalization  
- ER Diagram Design  

## Tools Used
- MySQL Workbench  
- dbdiagram.io  
- GitHub
